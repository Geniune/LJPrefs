* Apple Music 音乐

* Calendar 日历
* Camera 相机
* Contacts 联系人
* Health 健康
* Home 家庭
* Location 定位
* Microphone 麦克风
* Motion 运动与健身 
* Photos 相册
* Reminders 提醒事项
